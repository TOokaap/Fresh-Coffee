## Fresh Coffee

Ce projet est le site web de d'une entreprise fictive Fresh Coffee, ce site permet à l'entreprise d'exposer son contenu, ses articles, ses évènements prévus et/ou passés, sa gallerie photo, sa suggestion de plats/boissons, sa carte ainsi qu'un formulaire de réservation sur plan ou non, afin de permettre au client de passer une réservation.

## Stack

### Front-end :
- Next.js 16
- Typescript
- Tailwind CSS
